Visualization Temperature by Latitude

Understanding the weather and it's trends in a single location is an incredibly complex ontaking. Now expand on that by trying to identify global weather trends and you're going to run into challenges without some sound data and accompanying visualizations.

This weather analytics dashboard is a project that looked at over 500 randomly genearated locations throughout the world. Using Python and the OpenWeather API, the goal of this project was to understand how latitude impacts four key weather observations: Temperature, Humidity, Windspeed, and Cloudiness.
